SNAKE

Play Snake
Snake is a video game genre where the player maneuvers a growing line that becomes a primary obstacle to itself.

Creator: Gremlin Industries
Original Release: 1976

Instructions

  1. Use Up Arrow Key, Down Arrow Key, Left Arrow Key and Right Arrow Key to move the snake in the respective direction.
  2. There will be food for the snake to eat on the screen.
  3. For each food block you eat, the snake grows a little longer and you are given a point.
  4. If you collided into yourself, the game shall end.
  5. Have Fun!
