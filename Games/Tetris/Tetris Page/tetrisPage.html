TETRIS

Play Tetris
In Tetris, players complete lines by moving differently shaped pieces, which descend onto the playing field. The game ends when the playing field is filled. The longer the player can delay this outcome, the higher their score will be.

Creator: Alexey Pajitnov
Original Release: 1985

Instructions

  1. Try to arrange the blocks in a way that it prevents the blocks from stacking up to the top of the screen.
  2. Use the Left Arrow Key and Right Arrow Key to move your blocks.
  3. Use the Up Arrow Key to rotate the block.
  4. The block will only rotate if there is enough space on both sides.
  5. Have Fun!
